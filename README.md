# Log Analyzer (Rust)

Консольный инструмент для анализа и фильтрации больших файлов логов с прицелом на производительность.

## Основные возможности (в разработке)
*   🚀 Быстрое чтение файлов любого размера.
*   🔍 Фильтрация по уровню логирования (INFO, ERROR, WARN).
*   📊 Сбор статистики по ключевым событиям.

## Требования
*   Rust (последняя стабильная версия)
*   Windows / macOS / Linux

## Запуск проекта
Для запуска в режиме разработки используйте Cargo:

cargo run -- <путь_к_файлу>
